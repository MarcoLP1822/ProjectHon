We are going to create a simple web app that allows users to upload their book and get various types of content from the book and let us upload content directly on a social media platform.

The app will have the following features:
- User Authentication
- User Dashboard
- Upload Book
- Create Summary
- Generate Keywords
- Generate Categories
- Create Blog Post
- Create Podcast Script
- Create Social Media Post (generate a post and image with buttons to publish on instagram)
